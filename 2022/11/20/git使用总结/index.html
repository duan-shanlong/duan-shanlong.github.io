Git基本概念

Git 分布式版本控制系统 linus 2周 C语言写的

版本控制系统 version control system ==> VCS

作用： 代码存档备份，支持多人协作开发。

版本控制系统的分类

  1. 本地VCS
  2. 集中式VCS ==> SVN
  3. 分布式VCS ==> Git

Git 三个区域

  1. 工作区 workspace 红色
  2. 暂存区 staged 绿色
  3. Git仓库
